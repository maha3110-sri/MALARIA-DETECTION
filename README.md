# Autism Prediction Using Machine Learning

This project aims to predict Autism Spectrum Disorder (ASD) using
machine learning techniques based on behavioral and demographic data.
Early prediction can help in faster screening and awareness.

## Project Description
Autism Spectrum Disorder is a developmental condition that affects
communication, behavior, and social interaction. This project uses
machine learning classification algorithms to predict the likelihood
of autism traits from given input features.
